# Java 离线授权系统（Offline License System）

一个轻量级、无需联网的 Java 应用授权验证框架，适用于 Spring Boot 私有化部署场景。支持绑定机器指纹、有效期控制和功能模块授权。

> 💡 本项目为 **参考实现模板**，可自由用于个人或商业项目，但**不包含任何厂商私钥**。实际商用时需自行生成密钥对并集成到你的产品中。

## ✨ 特性

- ✅ **完全离线运行**：无需连接网络，适合内网/隔离环境
- ✅ **机器绑定**：基于 MAC 地址 + 主机名生成唯一指纹，防止随意复制
- ✅ **防篡改**：使用 RSA 非对称加密签名，确保 License 文件不可伪造
- ✅ **灵活控制**：支持设置授权客户、过期时间、功能模块等
- ✅ **Spring Boot 友好**：通过 `@PostConstruct` 自动验证，启动即生效
- ✅ **JDK 8 兼容**：仅依赖标准 Java API，无第三方库强制依赖

## ⚠️ 重要说明

本仓库提供的是 **通用授权框架的示例代码**，**并非开箱即用的完整授权解决方案**。  
若用于生产环境，请务必完成以下步骤：

1. **生成你自己的 RSA 密钥对**（不要使用示例密钥！）
2. **将你的公钥硬编码或安全注入到验证器中**
3. **对最终 JAR 包进行代码混淆**（如 ProGuard），防止反编译绕过
4. **在软件许可协议（EULA）中明确禁止反向工程与非法分发**

> 🔒 **切勿将你的私钥（`private.key`）提交到任何代码仓库！**
